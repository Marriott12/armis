# Help & FAQ

## Using the Analytics Dashboard

- **Sidebar Navigation:** Use the sidebar to quickly jump to different analytics sections. Press `Alt+S` to toggle the sidebar.
- **Filters:** Use the multi-select filters at the top to refine your results.
- **Drill-down:** Click on any chart bar to drill down into that dimension; use the breadcrumbs to go back.
- **Export & Save:** Use the Export & Save panel to export data as CSV, images, or save your dashboard state.
- **Notifications:** Subscribe to analytics notifications in the Notifications panel (if enabled).
- **API Access:** See the API Access panel for how to pull analytics data externally.

## FAQ

**Q: Why do I see fewer records than expected?**  
A: You may have filters applied (check above the table or in the breadcrumb trail).

**Q: Can I get automated reports by email?**  
A: If enabled by your admin, you can schedule notifications in the Notifications panel.

**Q: How do I use the API?**  
A: Go to the API Access panel for examples. Authenticated users can pull data for integration.

_For more guidance, contact your system administrator._
